## Analysis
The script has a **global** string `a` which contains the **alphabet** the program uses for **creating** strings.  
We also see some **functions** with non-descriptive names.  

The program starts towards the end
```python
arg444 = arg132()
arg432 = arg232()
arg133(arg432)
arg112()
arg423 = arg111(arg444)
print(arg423)
sys.exit(0)
```

When we **run** the program, we are asked for an **input** which is then checked.  
```
Please enter correct password for flag: test
That password is incorrect
```
So we find **where** the input happens in the code by searching for the `input()` function.  
We **find** it in function `arg232()`
```python
return input(a[47]+a[75]+a[68]+a[64]+a[82]+a[68]+a[94]+a[68]+a[77]+a[83]+\
a[68]+a[81]+a[94]+a[66]+a[78]+a[81]+a[81]+a[68]+a[66]+a[83]+\

a[69]+a[78]+a[81]+a[94]+a[69]+a[75]+a[64]+a[70]+a[25]+a[94])
```
The long **chain** of concatenated values of `a` form the above output string.  
Our input is stored in `arg432` which is then the argument of `arg133()`.  
This probably **checks** the password we gave
```python
def arg133(arg432):
  if arg432 == a[71]+a[64]+a[79]+a[79]+a[88]+a[66]+a[71]+a[64]+a[77]+a[66]+a[68]:
```

The **chain** of values from `a` in the `if` are probably the correct password.  
We add a **print** above the `if` and print it.
```python

```
And we get
```

```
So we try the password `happychance` and get our flag.
